npm - node package manager

npm install - instala um pacote npm
    global        - a nível da máquina
    local / projeto - a nível do projeto
    desenvolvimento - a nível de desenvolvimento

npm install -D nome-do-pacote
             cypress

npm install -D cypress

npm install -D cypress@13.7.3

rodar teste: npx cypress run

Triplo A - Arrange, Act e Assert

    Preparacao - Arrange 
      - acessando o site 
      - criando alguma massa de dados 
      - navegando para alguma página
    
    Acao - Act
      - preencher e salvar o formulario 
      - confirmar a exclusao

    Resultado esperado - Assert
      - visualizar a tela de cadastrar concluído
      - confirmar que a exclusao foi feita
