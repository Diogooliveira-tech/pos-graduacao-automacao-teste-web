# CSS Selectors

- Tag: `table`
- ID: `#balance`
- Classe: `.button.new`
- Atributo: `[onclick="Modal.open()"]`

## Coringas

- `^` inicia com valor: `[onclick^="Modal.o"]`
- `$` termina com valor
- `*` contém o valor

## Dedicados à testes

# XPath Selectors

- `//` - nós de um documento
- `*` - tudo ou todos
- `[@nome_atributo="valor_atributo"]` - sintaxe para atributos

### Exemplo de XPath:

- `/html/body/main/section[2]`
- `//*[text() = "Transações"]`
- `//*[contains(text(), "Saídas")]`
- `//*[contains(text(), "Nova")][1]`
